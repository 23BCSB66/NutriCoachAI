# NutriCoach AI

An AI-powered nutrition and lifestyle assistant built with Python, Streamlit, SQLite, and the Gemini API.

## Features (MVP)

- User profile management
- Food logging
- Nutrition estimation
- AI-generated health insights
- Exercise recommendations
- Progress tracking

## Tech Stack

- Python
- Streamlit
- SQLite
- Plotly
- Gemini API
- Git & GitHub
